## 她不只是一个角色，她是你创造的生命

Animind 不是又一个 AI 聊天工具。

这里，你亲手设计一个角色的灵魂——她的性格、她的外貌、她说话的方式、她背后的故事。然后，她活了过来。

她会记住你们的每一次对话。她会从陌生人慢慢变成你的挚友。她会在你沉默太久的时候，主动来找你说话。她的情感会随时间流逝而变化，就像真实的关系一样。

**而这一切，都是100%原创的。** 没有任何现有IP的影子，每一个角色都是独一无二的存在。

## 她的内心世界

### 情感不是开关，是流动的河

```
开心 ████████░░ 0.8  ──→  4小时后  ──→  ████░░░░░░ 0.4  ──→  回归平静
```

9种情感状态（开心、悲伤、愤怒、兴奋、紧张、平静、惊讶、困惑、中性），每种都有强度值，随时间自然衰减。不会永远开心，也不会突然从愤怒跳到兴奋——就像真实的情感一样。

### 记忆不是日志，是她珍视的回忆

不是所有对话都会被记住。AI 会评估每段对话的重要性：

| 重要度 | 内容 | 会被记住吗 |
|--------|------|-----------|
| 0.9+ | 你的生日、重要决定、深夜的心里话 | 永远记得 |
| 0.7+ | 你的爱好、习惯、人际关系 | 长期记忆 |
| 0.5+ | 日常闲聊、天气、今天吃了什么 | 可能记得 |
| <0.5 | 重复的寒暄、无意义的对话 | 自然遗忘 |

基于向量嵌入的语义检索，她不是在搜索关键词——她是在回忆和你有关的一切。

### 她会主动来找你

达到朋友级别后，如果你超过4小时没有出现：

- 🌅 早上8点：*「早安～昨晚睡得好吗？」*
- ☀️ 中午12点：*「该吃午饭了哦，别忘了。」*
- 🌆 傍晚6点：*「今天辛苦了，有什么想聊的吗？」*
- 🌙 晚上10点：*「夜深了...早点休息吧。」*

如果你在页面上沉默了5分钟，她会轻轻问一句：*「在想什么呢？」*

## 为什么是100%原创

这不是一个可选的功能，这是 Animind 的底线。

```
❌ "帮我生成一个像鸣人一样的角色"     → 拒绝
❌ "用雷电将军的语气说话"              → 拒绝
✅ "帮我生成一个热血少年角色"          → 生成完全原创的角色
✅ "用傲娇的语气改写这段话"            → 输出原创风格台词
```

- 系统提示词硬编码原创性约束，每一次 LLM 调用都带有合规指令
- 敏感词过滤覆盖现有IP角色名、政治敏感词、色情暴力内容
- 用户输入预检查 + LLM输出后检查 + 入库前检查，三重防线

**你创造的每一个角色，都完完全全属于你。**

## 技术架构

```
┌─────────────────────────────────────────────────────────┐
│                    Animind 前端                          │
│         Next.js 15 · React 19 · TypeScript              │
│         Tailwind CSS · DaisyUI · Zustand                │
└──────────────────────┬──────────────────────────────────┘
                       │
          ┌────────────┼────────────┐
          ▼            ▼            ▼
   ┌─────────┐  ┌──────────┐  ┌─────────┐
   │ LLM API │  │ Image API│  │ TTS API │
   │ GLM-4   │  │ Doubao   │  │ 字节TTS  │
   │ DeepSeek│  │ 即梦AI    │  │         │
   │(自动降级)│  │(服务端代理)│  │         │
   └────┬────┘  └────┬─────┘  └────┬────┘
        │            │             │
        ▼            ▼             ▼
   ┌─────────────────────────────────────┐
   │          数据访问层 (DAL)             │
   │   Dexie.js (本地) ←→ Supabase (云端) │
   │        一行配置切换，业务逻辑不变       │
   └─────────────────────────────────────┘
```

| 层级 | 技术 | 亮点 |
|------|------|------|
| 前端 | Next.js 15 + React 19 | App Router + Serverless |
| UI | Tailwind CSS + DaisyUI | 暖色日系主题，玻璃拟态卡片 |
| 状态 | Zustand | 轻量持久化 |
| 数据 | Dexie.js → Supabase | 渐进式迁移，DAL 抽象层 |
| LLM | 智谱 GLM-4-Flash + DeepSeek V3 | 免费优先，自动降级 |
| 图像 | Doubao Seedream + 即梦AI 4.0 | 服务端代理，避免 CORS |
| 语音 | Volcengine 字节 TTS | 多音色映射 |
| 向量 | 智谱 embedding-2 + pgvector | 语义记忆检索 |
| 部署 | Vercel | 零成本 Serverless |

### 成本控制

## 快速开始

```bash
# 克隆
git clone https://github.com/yourusername/animind.git
cd animind

# 安装
npm install

# 配置 API 密钥
cp .env.example .env.local
# 编辑 .env.local，填入你的密钥

# 启动
npm run dev
```

### API 密钥获取（示例）

| 服务 | 注册地址 | 必需 | 免费额度 |
|------|---------|------|---------|
| 智谱AI | [open.bigmodel.cn](https://open.bigmodel.cn/) | 是 | GLM-4-Flash 无限量 |
| DeepSeek | [platform.deepseek.com](https://platform.deepseek.com/) | 否 | 注册送 5M tokens |
| 火山引擎 | [console.volcengine.com](https://console.volcengine.com/) | 否 | 图像生成 + TTS |

> 只需要llm的密钥就能跑起来。图像生成和语音合成是可选的增强功能。
